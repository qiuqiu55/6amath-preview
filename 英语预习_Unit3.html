📘 语法讲解

可数与不可数名词

类别可数名词不可数名词
单数形式名词前加 a/an
e.g. buy a tomato, have an apple		名词前不加 a/an
e.g. drink milk
复数形式大多数结尾加 -s
e.g. want two cucumbers		没有复数形式
表示数量用数词或数量表达
e.g. three oranges, a bag of noodles,
many eggs, a lot of/lots of grapes		用数量表达
e.g. a bottle of orange juice,
not much yogurt, a lot of cheese
询问数量用 how many
e.g. How many carrots would you like?		用 how much
e.g. How much salt should I add?

1. Countable nouns are things we can count. 可数名词通常可计数。
e.g. There is an onion in the fridge. There are many onions in the fridge.

2. Uncountable nouns are things we cannot count or that are difficult to count. 不可数名词通常不能或难以计数。
e.g. How much water do we need to drink every day?

3. Some nouns are both countable and uncountable. 有些名词既可作可数名词，也可作不可数名词。
e.g. Would you like some salad? Two salads, please.

可数名词复数的拼写规则

名词类型复数形式示例
大多数名词+spepper → peppers
以 ch, sh, s, ss, x, z 结尾+espeach → peaches, dish → dishes
以 o 结尾+s 或 +eskilo → kilos, tomato → tomatoes
以"元音字母+y"结尾+sday → days
以"辅音字母+y"结尾去 y 变 i，再加 esstrawberry → strawberries
部分以 f/fe 结尾去 f/fe 变 v，再加 esknife → knives
其他名词词尾加字母child → children
改变元音字母man → men, tooth → teeth
单复数同形sheep → sheep, fish → fish

Note: We don't use "a little/much/many + of + noun". 直接在 a little, much 或 many 后加名词，不要加 of。
We should use "a little/much/many + of + the + noun". 如果加 of，则名词前需加定冠词 the。
